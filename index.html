Principais Projetos

Secret Word

Secret Word é um jogo de adivinhação de palavras desenvolvido com React, JavaScript e Bootstrap 5. Utilizando React para componentização e gerenciamento de estado, o jogo oferece uma experiência interativa e dinâmica. O JavaScript proporciona interações fluidas e eficientes, enquanto o Bootstrap 5 garante uma interface limpa e funcional. O projeto não só oferece entretenimento, mas também demonstra efetivamente a aplicação prática dessas tecnologias de ponta no desenvolvimento de aplicativos web.
